    Signal Detection Using Machine Learning


    This was a project that I developed during my master's degree. The goal was to detect signals between a large amount of background data. To measure how good the selection was (i.e hypothesis test), was used the significance as a score function.
    The first procedure was a standard high-energy procedure, and I used ROOT (C++) for the analysis. To improve the results, a second procedure was performed, a machine learning selection. The ML algorithm used to perform the selection was Gradient Boosted Decision Tree (XGBoost) which had 40 features. A lot of models were tested but the final one shows a result three times better than the standard procedure.

    GEANT4 Monte Carlo Simulation in C++



    This is a project that I developed during my undergraduate research in the Leptons Laboratory at UNICAMP. This simulation aims to simulate the apparatus used in the real experiment performed in the laboratory, the radioactive source used, and quantify how many photons arrive in the acceptance window of the X-ARAPUCA light detector. With this information, it is possible to combine it with experimental data and obtains the total efficiency of the device.
    The simulation was done in C and C++ using the GEANT4 package, in which it was necessary to work with object-oriented programming to obtain a simulation that described the entire detector geometry, the physical processes involved and the detector response. With this work I was able to co-author two published articles: 2020 JINST 15 C05045 , 2021 JINST 16 P11002. In addition, my results have been presented at several working conferences, including some held at CERN.
